What is Image & PDF Tool?

Image & PDF Tool is a free, browser-based utility designed to help individuals prepare images and PDF documents that meet the strict requirements of government portals, online application systems, and official document submissions in India and beyond.

From passport photos and signatures to ID proofs and thumb impressions — every government document upload has its own format, size, and dimension requirements. Our tool takes the guesswork out of the process by letting you compress, convert, create, and validate files directly in your browser, without uploading anything to a server.

Why We Built This

Millions of people struggle every year when uploading documents to government portals. Common errors include:

  • File size exceeds the allowed limit
  • Wrong image dimensions (e.g., passport photo must be exactly 200×230 px)
  • Unsupported file formats
  • Special characters in file names
  • DPI/resolution out of range

We built this tool to be the one-stop solution for all these problems — fast, free, and private.

Key Features

📦 Compressor

Reduce image size to under 100 KB or PDFs to under 1 MB.

🔄 Converter

Convert between JPG, PNG, GIF, WebP, AVIF, TIFF, BMP, HEIC, and PDF.

📷 Image Creator

Create passport photos, signatures, and custom-sized images.

📄 PDF Tools

Merge PDFs, convert images to PDF, and compress PDF files.

✅ Validator

Validate files against government portal rules and auto-fix issues.

🔒 100% Private

All processing happens locally. No file is ever uploaded to any server.

Technology

This tool is built with plain HTML, CSS, and vanilla JavaScript. It uses the following open-source libraries (loaded via CDN) for advanced format support:

  • jsPDF – PDF creation from images
  • PDF.js – PDF rendering and compression
  • pdf-lib – PDF merging
  • UTIF.js – TIFF image encoding and decoding
  • heic2any – HEIC/HEIF decoding for browsers

No frameworks. No tracking. No cookies. Just fast, private tools.
