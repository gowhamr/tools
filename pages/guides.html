Image Compression

How to Compress Images for Government Portals

Most Indian government portals (UIDAI, DigiLocker, passport.gov.in) require images under 50–200 KB. Here's how to hit that target without losing quality.

  1. Open the Compress tool and upload your JPG or PNG.
  2. Choose the Balanced preset — this targets ~72% quality, typically reducing a 1 MB photo to under 150 KB.
  3. Check the live file-size preview. If still too large, drag the Quality slider left toward Max Compress.
  4. Switch Output Format to JPG (most portals reject WebP).
  5. Click ⚡ Compress Now, then download.
💡 Tip: If the portal shows "File too large" even after compression, try switching to JPG at 50–60% quality. Colour photos usually still look fine at that level.

Format Conversion

How to Convert PNG to JPG (and Other Formats)

PNG files are lossless and often large. Converting to JPG or WebP can reduce size by 60–80% with no visible quality difference for photos.

  1. Open the Convert tool and upload one or more files.
  2. Under Convert to, pick your target format — JPG for compatibility, WebP for smallest size.
  3. Set Quality to 85% for photos (PNG screenshots can go lower).
  4. Click Convert Now and download.
ℹ️ Format guide:
JPG — best for photos & govt portals
PNG — best for logos & transparent images
WebP — smallest size, modern browsers only

Create & Resize

How to Create a Passport-Size Photo

Indian passport applications require a 200×230 px, white background, JPG photo under 200 KB. KaruviLab can resize and compress it in seconds.

  1. Open the Create / Resize tool.
  2. Click the Passport preset — this sets 200×230 px automatically.
  3. Upload your photo. Use a well-lit, front-facing shot on a plain background.
  4. Set Background to White and format to JPG.
  5. Click Create / Resize, then use the Compressor to bring it under 200 KB if needed.
✅ Accepted by: Passport Seva, UIDAI Aadhaar, DigiLocker, NEET/JEE applications, and most state government portals.
→ Open Create / Resize
Validation

How to Validate a Document Before Uploading

Avoid portal rejections by checking format, file size, and dimensions before you upload.

  1. Open the Validate tool and upload your file.
  2. Select the Document Type — Passport, Signature, ID Proof, or Thumb Print.
  3. Click ✓ Validate File.
  4. The tool checks: file format, size in KB, dimensions in px, and colour profile.
  5. Fix any flagged issues using Compress or Convert, then validate again.
📋 Common requirements checked: Passport Photo — JPG, 10–200 KB, 200×230 px, white background. Signature — JPG, 4–50 KB, 140×60 px.
→ Open Validate

PDF Creation

How to Convert Images to a Single PDF

Combine multiple scanned pages, photos, or screenshots into one PDF document — no software required.

  1. Open PDF Tools and select the Images → PDF tab.
  2. Upload all images. Drag to reorder pages if needed.
  3. Choose Page Size (A4 for official documents) and Orientation.
  4. Click 📄 Create PDF and download the result.
📌 Use case: Submitting multiple Aadhaar / PAN card sides as one PDF, or combining interview call letters into a single document.
→ Open PDF Tools
PDF Merge

How to Merge Multiple PDFs Into One

Many portals require all documents submitted as a single PDF. Merge them here instantly, without uploading to any cloud service.

  1. Open PDF Tools and select the Merge PDFs tab.
  2. Upload two or more PDF files.
  3. The files are merged in the order uploaded.
  4. Click 📋 Merge PDFs and download the combined file.
🔒 Privacy: All merging happens in your browser using pdf-lib. Your documents never touch a server.
→ Open PDF Tools
PDF Compression

How to Reduce PDF File Size

Scanned PDFs often exceed portal limits (1–5 MB). Our compressor recompresses the images embedded in the PDF to reduce size significantly.

  1. Open PDF Tools and select the Compress tab.
  2. Upload your PDF.
  3. Adjust the Image quality inside PDF slider — 60% is usually a good starting point.
  4. Click 🗜 Compress PDF and download.
⚠️ Note: This tool compresses image content within the PDF. Text-only PDFs (generated from Word, Excel, etc.) won't benefit — they're already small.

Base64

How to Encode & Decode Base64

Base64 is used in APIs, email attachments, data URIs, and JWT tokens. This tool encodes any text or decodes any Base64 string instantly.

  1. Open the Base64 tool.
  2. Select the Encode tab to convert plain text → Base64, or Decode to reverse it.
  3. Paste your input in the text area.
  4. Click ⚡ Convert and copy the result.
Common uses: Embed images in CSS/HTML as data URIs, decode JWT payloads, handle API authentication tokens, encode binary data for transmission.
→ Open Base64 Tool
Regex

How to Test Regular Expressions

Write, test, and debug regex patterns with live match highlighting. Includes presets for email, phone numbers, URLs, PAN cards, and more.

  1. Open the Regex Tester tool.
  2. Pick a preset (Email, IN Phone, PAN, etc.) or type your own pattern.
  3. Paste your test string in the Test Input area.
  4. Matches highlight in real time. Check the flags (g, i, m) as needed.
💡 Tip: Paste a full regex like /^[A-Z]{5}[0-9]{4}[A-Z]$/i directly into the Pattern field — the tool auto-extracts the flags.
→ Open Regex Tester
JSON / XML

How to Format & Validate JSON

Prettify minified JSON, minify formatted JSON for production, or validate a payload before sending it to an API.

  1. Open the JSON / XML tool.
  2. Paste your raw JSON into the input area.
  3. Select a mode: Format (pretty-print), Minify (compact), or Validate (check for errors).
  4. Click the action button. Syntax errors are highlighted with the exact position.
  5. Copy the output with one click.
✅ Runs entirely offline. Paste API keys, tokens, or sensitive JSON safely — nothing is sent to any server.

SIP Calculator

How to Calculate SIP Returns

A Systematic Investment Plan (SIP) lets you invest a fixed amount monthly in mutual funds. Use this calculator to estimate how much your investments will grow.

  1. Open Calculators → SIP Calculator.
  2. Enter your Monthly SIP amount (e.g. ₹5,000).
  3. Set Expected annual return. Index funds historically return 10–14% CAGR.
  4. Set the Investment period in years.
  5. View the projected corpus, total invested, and estimated returns instantly.
Example: ₹5,000/month × 12% p.a. × 15 years = ₹25.2 lakh estimated corpus on ₹9 lakh invested. That's 2.8× growth.
→ Open Calculators
EMI Calculator

How to Calculate Loan EMI

EMI (Equated Monthly Instalment) depends on loan amount, interest rate, and tenure. This calculator shows your monthly payment and total interest payable.

  1. Open Calculators → EMI Calculator.
  2. Enter the Loan amount (principal).
  3. Enter the Annual interest rate (e.g. 8.5% for a home loan).
  4. Set the Tenure in months or years.
  5. See the monthly EMI, total amount payable, and total interest instantly.
Formula: EMI = P × r × (1+r)ⁿ / ((1+r)ⁿ−1) where r = monthly rate, n = number of months.
→ Open Calculators
Date & Time

How to Convert UTC to IST (and Back)

Indian Standard Time (IST) is UTC +5:30. Use the converter when working with APIs, logs, or international meeting times.

  1. Open Calculators → UTC ↔ IST Converter.
  2. Select direction: UTC → IST or IST → UTC.
  3. Enter the date and time.
  4. The converted time is shown instantly with the ±5:30 offset applied.
Quick reference:
UTC 00:00 = IST 05:30
UTC 12:00 = IST 17:30
UTC 18:30 = IST 00:00 (next day)

❓ Frequently Asked Questions

Are my files safe? Do you store them?

Yes, completely safe. No files are ever uploaded to any server. Every tool — compression, conversion, validation, PDF merge — runs entirely inside your browser using JavaScript and the Canvas API. When you close the tab, everything is gone.

Why is my compressed image still too large for the government portal?

Try switching to JPG format (not PNG) and use a quality of 50–65%. Most portals have strict limits — UIDAI passport photo requires under 200 KB, many signature fields require under 50 KB. Use the Validate tool to check exact requirements before re-uploading.

Which image formats are supported?

Input: JPG, JPEG, JFIF, PNG, GIF, WebP, AVIF, TIFF, BMP, HEIC (on supported browsers). Output: JPG, PNG, GIF, WebP, AVIF, TIFF, BMP, PDF.

Does KaruviLab work offline?

Yes — after your first visit, the app is cached as a PWA. You can add it to your home screen and use all tools without an internet connection. Only the Calculators page requires a network connection on first load.

Why does my PDF compress only a little?

Our PDF compressor recompresses images embedded within the PDF. If your PDF was generated digitally (from Word, Excel, or a print-to-PDF), it contains very little image data and won't compress much. For scanned documents, expect 40–70% size reduction.

Can I compress multiple images at once?

Yes. The Compress tool accepts multiple files. Each is processed individually and shows its own before/after size comparison. You can download files one by one or all at once.

Is KaruviLab free?

Completely free, with no sign-up, no watermarks, and no limits on the number of files. The app is funded by the developer as a public utility for students, job seekers, and anyone dealing with government document submissions.
